## 需安裝套件
使用前要記得幫您的樹莓派安裝以下套件
```shell
sudo apt install python3-pip
pip3 install telegram
pip3 install picamera
pip3 install python-telegram-bot
pip3 install 
#.h264轉.mp4
sudo apt install gpac
```
記得要到 `raspi-config` 裡面將 pi camera enable 喔
